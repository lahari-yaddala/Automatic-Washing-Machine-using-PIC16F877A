# Automatic-Washing-Machine-using-PIC16F877A

# 🧺 Washing Machine Simulation using PIC16F877A

An IoT and Embedded Systems project developed during my **IoT Internship at Emertxe Solutions**. This project simulates the working of an automatic washing machine using the **PIC16F877A microcontroller**. The system was implemented in **Embedded C** using **MPLAB X IDE** and simulated in **PICSimLab**.

---

## 📌 Project Overview

This project demonstrates the automation of a washing machine using embedded systems. The PIC16F877A microcontroller controls the washing sequence, including water filling, washing, rinsing, draining, and spinning operations.

The project focuses on developing embedded control logic, peripheral interfacing, and real-time hardware simulation.

---

## 🚀 Features

- Automatic washing cycle simulation
- Water filling and draining control
- Washing and rinsing operations
- Spin cycle implementation
- Embedded C based control logic
- Real-time simulation using PICSimLab
- Microcontroller-based automation
